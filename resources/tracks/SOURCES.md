# Track Metadata Sources

No bundled sector-boundary dataset is currently included.

Reason: this project has not yet verified a redistribution-compatible source for ACC sector boundaries by track and layout. The application therefore uses direct ACC shared-memory timing first and keeps track-boundary fallback unavailable for unknown tracks instead of inventing inaccurate sector boundaries.

User overrides may be imported as JSON into the user settings directory. Overrides must provide normalized sector boundaries in strictly increasing order and are treated as user-supplied data.
